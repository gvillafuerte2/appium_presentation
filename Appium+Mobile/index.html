Instalación

  • Mac OSX 10.7+
  • XCode >= 6.1+ (iOS Simulator)
  • XCode Command Line Tools and XCTool
  • Instalar Appium utilizando Node.js
  • Ruby Version Manager, Bundler
  • Appium Ruby Console (ARC)
  • Appium Server (aplicación de escritorio)
